### UNIT 7 문자열 사용하기
#### 7-1 문자열을 변수에 저장하기

앞에서  x = 10과 같이 변수를 만들고 숫자를 할당 했었죠? 마찬가지로 변수에 문자열을 할당하면 문자열이 저장됩니다. 먼저 문자열 'Hello world!' 를 변수에 저장하고 출력해보겠습니다.

```
hello1 = 'Hello, world!'
print(hello1)
```

여기에서는 Hello, world! 를  ' '(작은따옴표)로 묶었습니다. 파이썬에서는 문자열을 사용할 때 주로 작은 따옴표를 사용합니다. 물론 " " (큰따옴표)로 문자열을 사용할 수도 있습니다.

```
hello2 = "Hello world!"
print(hello2)
```
